# `Project:Responder` Docs
This documentation describes the functionality and features of the `Project:Responder` code base. The docs cover the following areas: 
- Terraform
- Ansible
- Building
- Inventory
- Future Improvements

## Overview
`Project:Responder` is an initiative of CSIRT to up-skill Incident Responders. This repository contains all the code required to build and configure an environment for the training. It is designed to mimic that of a small corporate network with typical workloads and network segmentation.


## Future

### Current use cases
- Crowdstrike RTR Assessment
- Crowdstrike Training Environment
- Lab environment for various windows forensics training packages
- Lab environment for RTR Powershell zero-to-hero training package

### Future use cases
- On-demand attack simulations and accompanying training material
- SSRC Tool evaluations (Falcon Forensics, Intezer, etc.)
- Dev environment for Crowdstrike Slack bots / XSOAR testing
- Detection evaluation - Develop and test new detection techniques
- Hybrid TTX exercises - TTX with hands-on components
- Expansion to facilitate Hyperforce-specific on-demand training and practical exercises (Kubernetes/containers, etc.)
