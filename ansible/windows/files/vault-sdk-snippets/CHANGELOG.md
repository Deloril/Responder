# Changelog

All notable changes to the "vault-sdk-snippets" extension will be documented in this file.

## [1.2.0] - 2025-07-14

### Added
- Go language support for Vault SDK completions
- Ruby language support
- Inline documentation hover provider for vault.read/write/list/delete

### Fixed
- Completion provider not triggering in TypeScript files
- Status bar icon not updating on workspace change

## [1.1.0] - 2024-05-22

### Added
- Python language support (hvac library completions)
- Auto-update mechanism for seamless version updates
- Status bar indicator showing Vault SDK connection status

### Changed
- Improved snippet formatting for JavaScript/TypeScript

## [1.0.0] - 2024-01-08

### Added
- Initial release
- JavaScript and TypeScript Vault SDK completions
- Code snippets for common vault operations (read, write, list, delete)
- Command palette integration (Vault SDK: Lookup Secret, Vault SDK: List Secrets)
