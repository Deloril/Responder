--------------------------------------------------------------------------------
PURPOSE
--------------------------------------------------------------------------------
This checklist ensures consistent onboarding of new staff at The Secrets Group.
Given our role as a digital escrow and secret custodianship provider, security
clearance and compliance requirements are mandatory. All items must be
completed before vault or client data access is granted.

--------------------------------------------------------------------------------
PRE-START (Completed by HR before Day 1)
--------------------------------------------------------------------------------
[ ] Offer letter signed and returned
[ ] Employment agreement executed
[ ] Tax declaration (IR330) received
[ ] Bank details for payroll confirmed
[ ] Emergency contact form completed
[ ] Proof of identity verified (passport/driver licence)
[ ] Right to work verified (NZ/AU/SG as applicable)
[ ] Background check initiated (police vetting for vault roles)
[ ] References checked
[ ] Desk/workspace assigned
[ ] IT equipment ordered
[ ] Security access request submitted

--------------------------------------------------------------------------------
DAY 1 - SECURITY & COMPLIANCE
--------------------------------------------------------------------------------

[ ] Code of Conduct signed
[ ] Conflict of Interest declaration completed
[ ] Insider trading policy (if applicable) acknowledged
[ ] Escrow Custodian Oath administered (vault roles only)
[ ] Photo ID badge issued
[ ] Building access card activated

[ ] Password policy explained
[ ] Clean desk policy explained
[ ] Visitor/escort procedures explained

--------------------------------------------------------------------------------
SECURITY CLEARANCE TIERS
--------------------------------------------------------------------------------
Tier 1 - General: Office, email, shared drives. No vault access.
Tier 2 - Restricted: As Tier 1 + read-only vault metadata. No secret content.
Tier 3 - Full: As Tier 2 + vault content access per client authorisation.
Tier 4 - Custodian: Full vault + dual-control authority. Police vetting required.
