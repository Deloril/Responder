7. RECOVERY PROCEDURES (SUMMARY)
--------------------------------------------------------------------------------
7.1 Vault Data Recovery
   - Mount Veeam backup; restore to alternate location if primary unavailable
   - Verify checksums; notify Tama Thomas (Vault Custodian) for validation
   - Document recovery in incident log

7.2 SQL Recovery
   - Restore from latest full + differential + log sequence
   - Point-in-time recovery available within 15-min RPO
   - Update connection strings if restored to alternate server

7.3 File Server Recovery
   - Restore from Veeam; verify share permissions
   - Replicate from Azure if local backup corrupted
