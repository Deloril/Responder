1. INTRODUCTION
---------------
The Secrets Group (TSG) invites proposals from qualified vendors for the
supply and installation of data centre equipment to refresh our Wellington
primary facility. This refresh supports the SSN platform roadmap, cloud
migration Phase 2, and vault expansion initiatives.

3. REQUIREMENTS
---------------

3.1 COMPUTE
- Minimum 16 physical hosts (2U preferred)
- 2x CPUs per host, 16+ cores each
- 256GB RAM minimum per host
- 4x 25GbE NICs per host
- Redundant PSUs
- 5-year warranty with 4-hour response
- Compatibility: VMware vSphere 8.x, Windows Server 2022, RHEL 9

3.2 STORAGE
- Primary: All-flash array, minimum 200TB usable (after RAID)
- Secondary: Hybrid or HDD for backup target, 300TB usable
- Replication capability for Sydney vault (future)
- Snapshots, thin provisioning
- 5-year warranty, same-day parts

3.3 NETWORKING
- 2x spine switches (redundant), 100GbE capable
- 2x leaf switches for server connectivity
- 25GbE server-facing ports
- BGP/OSPF support for future multi-site
- NX-OS or equivalent enterprise OS

3.4 BACKUP
- Deduplication appliance, 100TB usable
- Integration with Veeam Backup & Replication
- Replication to secondary site (Wellington to Auckland DR)
- 5-year warranty
