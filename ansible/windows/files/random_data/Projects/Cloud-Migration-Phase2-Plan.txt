PROGRAMME OVERVIEW
------------------
Phase 1 of TSG's cloud migration (completed Q4 2025) successfully migrated the
SSN web portal and customer-facing APIs to Microsoft Azure. Phase 2 extends
migration to secondary systems: email, collaboration, and development
environments.

PHASE 1 COMPLETED (Q4 2025)
---------------------------
- SSN Web Portal: Azure App Service (Wellington region)
- Customer API Gateway: Azure API Management
- Static assets: Azure Blob Storage + CDN
- DNS: Azure DNS for public-facing domains

SECURITY ARCHITECTURE
--------------------
- Conditional Access: MFA required for all cloud services
- Azure AD Connect: Hybrid identity (TSG-INTERNAL.LAB sync)
- DLP policies: Prevent external sharing of "Confidential" classified docs
- Encryption: Data at rest (AES-256), in transit (TLS 1.3)
- Audit logging: All admin actions to Azure Log Analytics
- Vault systems: REMAIN ON-PREMISES (out of scope for Phase 2)

RISKS & MITIGATIONS
-------------------
- Email downtime: Staged migration, weekend cutover, rollback plan
- SharePoint large files: Pilot with IT team first; throttle migration
- Dev environment drift: Infrastructure as Code (Bicep) for consistency
